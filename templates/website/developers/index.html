GovTrack applies the principles of open data to legislative transparency. We screen scrape a variety of official government websites each day and make the resulting normalized database of legislative information available for free for reuse — in bulk (since 2005) and with an API (since 2012). Our database is the foundation for dozens of other open government websites.

How to get the data

Starting in 2013 we began transitioning to a newer data collection system, so the data is currently fragmented among several sources:

github:unitedstates/congress-legislators. This is a community project that we started to maintain a comprehensive database of all Members of Congress since 1789. The github account contains YAML files which you can load into your application. (Before 2013, this was our "people" database.)

github:unitedstates/congress. This is a community project to compile a comprehensive database of U.S. federal legislation. It is based in part on the source code we developed in the early 2000’s to scrape THOMAS, and now we maintain the new code with others as part of a group effort. The project provides bulk data downloads, or you can run the Python scripts yourself on your own schedule.

Legacy Bulk Data. Since 2005 we have provided our database of flat XML files for bills and related data via our bulk data rsync server. This is the easiest method to get the complete database of bill status data and to keep it updated. Data files for the 113th Congress are from the two github projects above, so the information is identical. Historical data is from our legacy scrapers.

API. Our API went online in May 2012. It is not quite as complete as some of the other data access methods. Data for the 113th Congress are from the two github projects above, so the information is identical. Historical data is from our legacy scrapers.

GIS API and Maps. We host a map server and API for United States Congressional Districts, 113th Congress (i.e. 2012 elections and on), based on the geographic data made available by the U.S. Census.
